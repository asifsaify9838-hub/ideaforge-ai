# IdeaForge AI - Final (Full OpenAI setup)

This project is a Next.js (App Router) starter that includes a dashboard UI and API routes
to generate script, scenes, music prompts, and TTS via ElevenLabs (example).

IMPORTANT:
- Add your OpenAI API key and ElevenLabs key as environment variables on Vercel:
  OPENAI_API_KEY, ELEVENLABS_API_KEY
- Replace public/logo.png and public/bg.jpg with your real images if needed.
